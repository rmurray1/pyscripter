*********** Esta es la versión de 64 bits de PyScripter. ************
Requiere una versión de Windows de 64 bits. Tenga en cuenta que PyScripter
solo funciona si está instalada una versión de Python de 64 bits.

Antes de usar PyScripter, debe asegurarse de tener instalada en su maquina
una versión de Python mayor o igual a 3.7. Si no, puedes descargar uno de
http://www.python.org/.

Visite https://github.com/pyscripter/pyscripter/wiki para obtener información
sobre depuración remota y otros temas relacionados con PyScripter.

También puede verificar si hay una versión más nueva de PyScripter en
https://sourceforge.net/projects/pyscripter/files/.

Gracias por su interés en PyScripter.
